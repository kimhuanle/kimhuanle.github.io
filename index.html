About Me

My name is Huan Le. I am a Vietnamese international student at Miami University. I am in my 4th year pursuing Electrical Engineering Bachelor's degree, with a minor in Computer Science and Mathematics.

I love coding and solving complex problems. I am an easy going person and team player. I love helping other people when I have a chance. If you are working on a project and need a programmer, please feel free to contact me.

Things I've Built

DatCord

This is a simple chat box inspired by Discord platform. The application had the capability to host multiple rooms. Users can choose to join a room and chat with other users in the same room. Each room is unique.The application was built using Node.js, Socket.io, and MongoDB..

MineSweeper

This is an adaptation of the classics Minesweeper game. The design is inspired by the Minesweeper game of Google. It was a fun experience building this game as I learned a new algorithm called flood fill. The game is built entirely using HTML, CSS, and JavaScript.

Weather Forecast

This was a project of my CSE 383 Web Programming class in Fall 2020. I found it very interesting and educative. In this project, I learned to build a RESTful application that can utilized a weather api to output information to the screen.

Flappy Bird

This is my first ever game built using Java. The game is based on a very famous game released in 2016 by Nguyen Ha Dong. It got so popular that he had to take it down from app store several weeks after release due to the pressure from the government. Anyway, it is one of my proudest project since it was the first ever game I built.
